# EcoTrack — Sustainable Living Community

**Live Site:** [EcoTrack](https://ecotrack-auth-9303d.web.app/)


EcoTrack is a community-driven platform that encourages eco-conscious individuals to participate in sustainability challenges, track their environmental impact, share tips, and discover local green events. The platform focuses on measurable environmental improvements and fostering a collaborative eco-community.

---

## Features

- **User Authentication:**  
  - Sign up and login via Email/Password or Google.  
  - Protected routes for personal dashboards and activities.  

- **Sustainability Challenges:**  
  - Browse and join ongoing challenges.  
  - Track progress and see measurable impact (e.g., CO₂ saved, plastic reduced).  
  - Advanced filtering by category, date, and participant count.  

- **Community Tips & Events:**  
  - Share eco-friendly tips and upvote contributions.  
  - Discover upcoming local green events with details and participation tracking.  

- **Dashboard & Activity Tracking:**  
  - Personal dashboard showing challenges joined and progress.  
  - Visualize individual impact and contributions.  

- **Responsive and Accessible UI:**  
  - Clean, modern design with consistent styling.  
  - Fully responsive for desktop, tablet, and mobile devices.  
  - Accessibility-friendly with semantic HTML, focus states, and alt text for images.  

- **Interactive & Dynamic UX:**  
  - Hero carousels, skeleton loaders, and global spinners for smooth data loading.  
  - Styled toast notifications for success and error messages.  
  - Error boundaries for runtime issues and a 404 page for unknown routes.  

---

## Tech Stack

- **Frontend:** React.js, Firebase Authentication, Axios, React Router, Styled Components/Chakra UI/Your choice  
- **Backend:** Node.js, Express.js, MongoDB  
- **Hosting:** Netlify/Surge (Client), Vercel (Server)
